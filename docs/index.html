📄 Project Overview

An IoT-based system to detect landslides and earthquakes using ESP32, moisture, vibration, and gyroscope sensors. It triggers alerts via buzzer and mobile app using Blynk.

  • Remote real-time monitoring with mobile app
  • Low-cost, easy to deploy in remote areas
  • Scalable and customizable
